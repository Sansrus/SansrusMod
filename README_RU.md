# SansrusMod

Клиентский мод для Minecraft, добавляющий различные улучшения качества жизни для улучшения игрового процесса.

[English version](README.md)

## Возможности

### Управление смертью
- **Метки смерти** - Автоматически создаёт кликабельное сообщение в чате с координатами смерти. Нажмите, чтобы создать метку в Xaero's Minimap
- **История смертей** - Сохраняет снимок вашего инвентаря в момент смерти, включая дату и время

### Инвентарь и предметы
- **Перенос предметов с Shift** - Удерживайте Shift и проводите мышью по слотам для быстрого переноса предметов
- **Выброс всех предметов с Ctrl** - Удерживайте Ctrl при выбросе предмета, чтобы выбросить все предметы этого типа из инвентаря
- **Подсветка стаков** - Подсвечивает слоты инвентаря с одинаковыми предметами
- **Компоненты предмета в тултипе** - Отображает компоненты предмета в тултипе для отладки

### Функции чата
- **Копирование сообщений из чата** - Shift + клик по сообщению копирует его в буфер обмена
- **Ввод команд кириллицей** - Пишите команды на русской раскладке - они автоматически конвертируются в латиницу
- **Парсер координат** - Парсит координаты из чата и создаёт метки Xaero по клику
- **Антиспам** - Группирует одинаковые сообщения в чате вместе

### Книги
- **Перелистывание книги зажатием** - Удерживайте кнопку перелистывания для автоматического переключения страниц
- **Скорость прокрутки книги** - Настраиваемая скорость автоматического перелистывания (1-40)

### Интерфейс и отображение
- **Слоты брони в хотбаре** - Показывает слоты брони в хотбаре (настраивается: выкл, по времени или всегда)
- **Тултип карты** - Предпросмотр карт в их тултипе без необходимости брать их в руки

### Геймплей
- **Отключение кулдауна ПКМ** - Снимает внутренний кулдаун правой кнопки мыши при зажатии дольше 20 тиков
- **Защита жителей** - Предотвращает случайную атаку жителей
- **Рендер элитр на нагруднике** - Отображает элитры на нагрудниках с тегом `glider`
- **Отмена виньетки укачивания** - Убирает эффект виньетки укачивания (используйте при чёрном экране с эффектом тошноты)

## Настройка

Откройте меню настроек через ModMenu в игре. Все функции можно включать и отключать индивидуально.
